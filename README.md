# Burrow

Burrow is a web-based tool for keeping notes on tunnels and pivots between hosts.

## Getting Started

### Prerequisites

```
node
```

### Installing

```
git clone https://github.com/jesseclvrt/burrow
cd burrow
npm install
```

### Development

```
npm start
```

## Built With

-   [typescript](https://www.typescriptlang.org/) - Language
